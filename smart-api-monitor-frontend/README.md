# Frontend Workspace

The deployable frontend for this monorepo lives in `smart-api-monitor-frontend/front-saas`.

## What is here

- `front-saas`: the only frontend application that should be developed and deployed

## Notes

The legacy root-level Next.js scaffold was removed to avoid duplicate apps, hardcoded local URLs, and deployment confusion.

If you are working on the UI, start in `front-saas` and ignore the parent folder except for this navigation note.
